TOML ↔ YAML 変換

TOML形式とYAML形式を相互に変換します

TOML/YAML変換とは？

TOML形式とYAML形式を相互に変換するツールです。設定ファイルの形式移行に便利です。

こんな時に便利

  • 設定ファイルの移行
  • Cargo.tomlの確認
  • GitHub Actions設定
  • 設定形式の統一

このツールはブラウザ上で完全に動作します。入力データがサーバーに送信されることはありません。
