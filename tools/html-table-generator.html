HTML テーブル ジェネレーターとは？

            
スプレッドシートのようなGUIでHTMLテーブルを作成し、対応するHTMLコードを自動生成するツールです。CSSスタイル付きのテーブルをすぐに使えます。

            
    
                
  • 行・列の追加削除、セル内でのテキスト編集
    • 
                
  • CSVを貼り付けて一括インポート
    • 
                
  • ヘッダー色・ボーダー・ストライプなどスタイル設定
    • 
                
  • インラインCSS付きでどんな環境でも即利用可能
