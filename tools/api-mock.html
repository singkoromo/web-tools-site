APIモック・レスポンス生成ツールとは？

フロントエンド開発でバックエンドAPIが未完成の際にダミーレスポンスを生成するツールです。HTTPメソッド・ステータスコード・ヘッダー・ボディを自由に設定できます。

  • 成功・エラー・リストなど用途別テンプレートを搭載
  • 模擬レイテンシで実際のAPIに近い挙動をシミュレート
  • 生成されたcURLコマンドやfetch()コードをコピー可能
  • すべてブラウザ内で完結・サーバー通信なし
