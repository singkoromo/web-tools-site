Apache VirtualHost設定ジェネレーターについて

Apache HTTP ServerのVirtualHost設定(.conf)をGUIで生成するツールです。HTTP・HTTPS(SSL/TLS)・リバースプロキシ・セキュリティヘッダー・Gzip圧縮など多彩な設定に対応しています。

静的サイト・PHPアプリ・リバースプロキシ・WordPressの4種類のプリセットを用意しています。生成した設定は/etc/apache2/sites-available/に保存し、a2ensiteコマンドで有効化してください。

SSL証明書の取得にはLet's Encrypt (Certbot)の利用を推奨します。証明書取得後、自動的に設定ファイルが更新される場合があります。
