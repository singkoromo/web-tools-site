JSONPathテスター

よく使うパス例:
$ (ルート) $.store.book $.store.book[0] $.store.book[*].author $.store.book[?(@.price<10)] $..author $..price
入力 JSON
抽出結果
JSONPath構文リファレンス
式説明
$ルートオブジェクト
.key子要素アクセス
[n]配列のn番目の要素
[*]配列の全要素
..再帰的に子孫を検索
[?(@.key==value)]フィルタ式
[start:end]配列スライス

JSONPathテスターとは？

JSONPath式を使ってJSONデータから値を抽出・検索するツールです。APIレスポンスの特定フィールドを取得する際に便利です。

こんな時に便利

  • APIレスポンスからの値抽出
  • JSONPath式の学習・検証
  • 複雑なJSONからのデータ検索
  • jqコマンドの事前テスト

このツールはブラウザ上で完全に動作します。入力データがサーバーに送信されることはありません。
