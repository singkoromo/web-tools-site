Git Diffビューアー

git diffの出力（unified diff形式）を貼り付けると、追加=緑・削除=赤でビジュアル表示します。

Git Diffビューアーとは？

git diffコマンドの出力（unified diff形式）をビジュアルにハイライト表示するツールです。GitHubのコードレビュー画面のような見た目で差分を確認できます。

使い方

  1. git diff または git diff HEAD~1 の出力をコピー
  2. テキストエリアに貼り付けて「表示」をクリック
  3. 追加行（緑）・削除行（赤）がビジュアル表示されます

すべてブラウザ上で動作します。入力データがサーバーに送信されることはありません。
