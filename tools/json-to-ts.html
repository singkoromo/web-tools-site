JSON → TypeScript 型定義生成

JSONデータからTypeScriptのinterface / type定義を自動生成します

JSON → TypeScript 型定義生成とは？

JSONデータを貼り付けるだけで、TypeScriptのinterface定義を自動生成します。APIレスポンスや設定ファイルの型定義作成に最適です。

主な機能

  • ネスト構造の自動処理（再帰的interface生成）
  • 配列型・null型の正確な推論
  • interface / type エイリアスの切り替え
  • オプショナルプロパティ・readonlyの付与
  • export キーワードの自動付与
