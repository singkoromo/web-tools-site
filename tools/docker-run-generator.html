Docker Runコマンドジェネレーターとは？

docker runの複雑なオプションをGUIで設定してコマンドを自動生成するツールです。ポートマッピング・環境変数・ボリュームマウントなど多彩なオプションに対応しています。

すべてブラウザ上で動作します。入力データがサーバーに送信されることはありません。
